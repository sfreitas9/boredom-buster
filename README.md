# Boredom Buster

App to help you find something to do when you are bored
---
## To install:
```
npm install
```
---
## To run:
```
npm run build
npm start
```
Then go to http://127.0.0.1:3000/ or http://localhost:3000 in your browser.
---
## To run tests:
```
npm test
```
